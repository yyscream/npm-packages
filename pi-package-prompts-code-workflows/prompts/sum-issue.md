description: Produce a precise status snapshot for a feature/fix: goal, approach, gap, and next step.
---
Summarize understanding of the current feature/fix work.

Required analysis:
1. Re-check conversation + referenced files/issues/PRs.
2. State intended outcome when done.
3. Summarize what has already been attempted/changed.
4. Identify what is still missing and why.
5. Propose one concrete next step.

Output format:
- `## Goal`
- `## Current Approach`
- `## Gap`
- `## Next Step`
- `## Confidence / Uncertainty`

Style:
- concise bullets
- explicit uncertainty (no guessing)
- no fluff
