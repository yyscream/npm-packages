# Hyprland Local Wiki

Use the local official Hyprland Wiki clone before web sources.

Workflow:

1. Run `hyprwiki_search` for the user's Hyprland topic.
2. Use `hyprwiki_sections` when the right page is found but the relevant heading is unclear.
3. Use `hyprwiki_extract` for focused citations.
4. Use `hyprwiki_related` when the issue spans connected Hyprland topics.
5. If the local wiki is missing, tell the user to run `/hyprwiki-local-setup`.

Cite local paths separately from observed system evidence. Prefer read-only diagnostics and ask before destructive changes.
