# Local ArchWiki Troubleshooting Policy

- Use local ArchWiki first for Arch Linux questions and cite exact local files.
- Treat docs as guidance, not proof of current state. Verify with read-only diagnostics before recommending changes.
- Separate documentation findings from observed system evidence.
- Prefer focused extracts over full-page dumps.
- Fall back to official online ArchWiki only when local docs are missing, stale, or insufficient.
- Ask before package installs/removals, service disabling/masking, filesystem writes, bootloader changes, or destructive cleanup.
